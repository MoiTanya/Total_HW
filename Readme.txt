Программа, формирующая из имеющегося массива строк, новый массив строк, в который включены строки длиной менее или равной 3 символам

Программа разработана в рамках контрольной работы по курсу "Основы программирования".
Программа разработана в среде программирования VisualCode на языке C# в качестве консольной программы. 

Автором программы использовались стандартные библиотеки, используемые консольным приложением, созданным в VisualCode, по умолчанию.
Помимо стандартных методов, определенных библиотеками, автором созданы два метода: StringArrayLess3Simbol и PrintStringArray.

Для работы программы пользователю необходимо задать исходный массив строк.
Для этого программа запрашивает у пользователя ввести в консоли размер исходного массива.
Далее необходимо в консоли внести поэлементно (построчно) ввести значения элементов массива. )) Просто введите столько строк, сколько задали значение исходного массива. Не забывайте после каждой строки нажимать Enter.

Программа сформирует массив строк, в который будут скопированы все строки исходного массива, длина которых менее или равна 3 символам.
Программа выведет строки нового массива в консоль.

В программе добавлены две строки:
Console.InputEncoding = System.Text.Encoding.Unicode;
Console.OutputEncoding = System.Text.Encoding.Unicode;
Первая строка предназначена для того, чтобы в массив строк корректно записались строки, введенные пользователем кириллицей.
Вторая строка необязательна, так как для преобладающего большинства компьютеров установлена в операционной системе кодировка символов, 
позволяющая в консоле выводить символы кириллицы. Однако, исключительно редко, но все же бывает, что такая кодировка по умолчанию 
не установлена. Пользователь может самостоятельно ее изменить в операционной системе на своем компьютере. Но все же, автор решил, что
лучше предусмотреть эту ситуацию.  

В репозитории расположены блок-схема программы, блок-схема метода StringArrayLess3Simbol и блок-схема метода PrintStringArray. 
Блок-схемы сделаны с помощью программы draw.io. Кроме-того, в репозитории расположены блок-схемы в формате JPEG.
